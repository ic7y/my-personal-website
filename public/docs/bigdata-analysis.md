## 项目大作业选题
### 题目一：电商数据分析系统
功能：选择一个你喜欢的互联网公司产品，针对这个产品设计业务报表统计指标已监控业务整体情况。
系统要求实现以下功能：
1.	小组成员介绍页面。
2.	登陆、注册功能。
3.	报表指标页面展示 。
分步实施：
1.	初步完成总体设计，搭好框架，设计交互菜单界面。
2.	完成账号注册、登录功能
3.	完成小组成员介绍页面。
4.	展示报表指标页面。

要求：
1.	计算结果保存在mysql数据库，供后续查询。
2.	利用express框架构建后端服务；
3.	前端展示页面技术不限 ；
4.	界面友好（良好的人机交互），程序必须有注释。

### 题目二：校园图书借阅数据分析系统
功能：选择校园图书借阅场景，针对校园图书借阅数据设计业务报表，以监控热门图书、借阅高峰时段、借阅人偏好等业务整体情况。
系统要求实现以下功能：
1.	小组成员介绍页面（独立完成无需此页面）。
2.	登录、注册功能。
3.	报表指标页面展示（含热门图书、借阅高峰、类别分布）。
分步实施：
1.	初步完成总体设计，搭好框架，设计交互菜单界面。
2.	完成账号注册、登录功能。
3.	完成小组成员介绍页面（独立完成无需此步骤）。
4.	完成数据获取（模拟生成/申请校园脱敏借阅数据），展示报表指标页面。
要求：
1.	计算结果保存在MySQL数据库，供后续查询。
2.	利用Sqoop、Hive、MySQL完成数据导入、处理与存储；
3.	前端展示页面可使用ECharts，界面友好（良好的人机交互）；
4.	程序必须有注释，核心代码注释清晰。
